# Data Annotation Quality Tracker

## Overview
Customer message annotation pipeline with 
quality tracking and SLA compliance measurement.

## What I Did
- Created 50 real customer service messages
- Manually annotated into 4 categories:
  COMPLAINT, INQUIRY, URGENT, FEEDBACK
- Built Python tracker measuring daily productivity
- Achieved 100% SLA compliance
- Generated quality report automatically

## Tools Used
- Python
- Pandas
- Microsoft Excel

## Results
- Total Messages    : 50
- COMPLAINT        : 6
- INQUIRY          : 17
- URGENT           : 14
- FEEDBACK         : 13
- Accuracy Rate    : 100%
- SLA Status       : MET ✅
